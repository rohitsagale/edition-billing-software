# Edition Films & Photography – Billing Software

A complete, professional billing and booking management system for a films & photography shop. Built with **Flask (Python)**, **React.js + Material-UI (MUI)**, and **MySQL**.

## ✨ Features

- 🔐 **Admin Login** (JWT authentication)
- 📦 **Product / Service Management** (photography packages, albums, prints, etc.)
- 👥 **Client Management** (store client details, phone, email, address)
- 📅 **Booking Management** with event types (Wedding, Engagement, Birthday, Corporate, etc.)
- 🧾 **Billing System** – generate bills linked to bookings
- 📊 **Dashboard Analytics** – total clients, bookings, revenue, monthly chart, top clients
- 🖨️ **Print Invoice** from bill details modal
- 🔄 **Auto‑update booking balance** after billing
- 📱 **Responsive UI** with Material‑UI sidebar layout

## 🛠️ Tech Stack

| Layer       | Technology                                                                 |
|-------------|----------------------------------------------------------------------------|
| Backend     | Flask, Flask‑SQLAlchemy, Flask‑JWT‑Extended, Flask‑CORS, PyMySQL          |
| Frontend    | React 18, Vite, Material‑UI (MUI), Axios, React Router DOM, Recharts      |
| Database    | MySQL                                                                      |
| Date Picker | MUI X Date Pickers + DayJS                                                 |
